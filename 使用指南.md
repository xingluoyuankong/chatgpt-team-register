## 🔧 常见问题

### Q: 提示"代理连接失败"？

**A: 检查本地代理是否启动：**
- Clash：确保已开启 System Proxy
- V2Ray：确保正在运行
- Shadowsocks：确保已启用本地代理

### Q: 如何确认代理端口？

**A: 打开代理软件查看：**
- Clash → Settings → Port
- V2Ray → 参数设置 → 本地端口
- Shadowsocks → 选项设置 → 本地代理

### Q: 还是不行？

**A: 手动创建 proxy.txt**
```
http://127.0.0.1:你的端口
```
